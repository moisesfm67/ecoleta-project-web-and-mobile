## Setup

### To work the web and mobile application, the api has to be running

### API

- run `npm install`
- run `npm run knex:migrate`
- run `npm run knex:seed`
- run `npm run dev`

### UI

- run `npm install`
- run `npm run start`

### MOBILE

- run `npm install`
- run `expo start`
